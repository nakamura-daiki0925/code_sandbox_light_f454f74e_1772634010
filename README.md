## ⚙️ アニメーション

- **スクロールリビール**：Intersection Observer APIによる `reveal` クラスのフェードアップ
- **FAQアコーディオン**：max-height トランジション
- **ヒーローパーティクル**：Canvas API による軽量白色パーティクル
- **ハンバーガーメニュー**：スライドイン/アウト
- **ヘッダー**：スクロール検知でシャドウ追加
- **カウントアップ**：`data-count` 属性で数字アニメーション
- **ホバーエフェクト**：カード浮き上がり・ボタン遷移

## 📁 ファイル構成

```
index.html          ← メインLP（全セクション）
css/
  style.css         ← カスタムCSS（フォント定義・アニメーション・FAQ等）
js/
  main.js           ← インタラクション・アニメーションJS
fonts/              ← LINESeedJPフォントファイル配置場所（要追加）
  LINESeedJP-Thin.woff2
  LINESeedJP-Regular.woff2
  LINESeedJP-Bold.woff2
  LINESeedJP-ExtraBold.woff2
images/             ← 画像素材配置場所（要追加）
  logo.svg          ← ロゴ
  nobi-ton.png      ← のびとんキャラクター
  hero-cart.png     ← ヒーローカート画像
  laptop.png        ← PCモックアップ
  ...
```

---

## 🔧 WordPress実装時の注意点

1. **フォントファイル**：`fonts/` フォルダに LINESeedJP の woff/woff2 ファイルを配置
2. **画像素材**：実際の画像が揃い次第、プレースホルダー部分を差し替え
3. **動画**：「1分でわかる」エリアに YouTube 埋め込みまたは `<video>` タグを設置
4. **お問い合わせ**：Contact Form 7 等のフォームURLに差し替え
5. **コラム/お知らせ**：WordPress投稿と連携する場合は PHP ループで出力
